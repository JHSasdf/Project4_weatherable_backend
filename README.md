## 시작 가이드
**REQUIREMENTS**
For building and running the application you need:
* Spring Boot 3.24 or higher

* **Installation**
```
$ git clone https://github.com/dydgjs200/Weatherable_backend.git
gradle build
Execute Weatherable Application
```
